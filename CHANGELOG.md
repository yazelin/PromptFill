# 更新日誌 (Changelog)

## [0.4.1] - 2025-12-12

### 新增 ✨
- 匯出圖片添加模糊背景效果，提升視覺質感
- 匯出底部浮水印區域增加版本號（動態獲取）和網址二維碼
- 行動端支援 Web Share API 儲存圖片到相簿
- 匯入時使用友好的 Toast 提示和自訂對話框
- 完整備份匯入時顯示詳細資料統計資訊

### 改進 🚀
- 匯出圖片格式從 PNG 改為 JPG（品質 92%），檔案大小減少 60-70%
- 匯出卡片重新佈局：圖片置頂（最高 300px）、標題精簡、版本號移至底部
- 匯出按鈕狀態顯示「匯出中...」，防止重複點擊
- 優化手機端頂部標題和按鈕佈局，減少擁擠感
- 行動端隱藏版本號和重新整理按鈕，優化圖示尺寸
- 移除匯入流程中的 `alert()` 和 `window.confirm()`

### 修復 🐛
- 修復圖片 URL 失敗後無法替換的問題
- 修復手機端匯入範本「選擇檔案後沒有動靜」的問題
- 圖片載入失敗時顯示 300×300 占位符

### 移除 ❌
- 匯出圖片底部移除 Prompt Template 標籤，簡化資訊展示

---

## [0.4.0] - 2025-12-10

### 新增 ✨
- 範本列表支援瀑布流展示
- 新增標籤過濾功能
- 支援匯入/匯出範本與詞庫（Beta）
- 新建範本支援上傳本地圖片或圖片 URL
- 新增「經典場景微縮復刻」和「可視化企業成長之路」範本
- 補充公司、畫幅比例、渲染風格（3D 像素）等詞庫

### 改進 🚀
- 預設使用本地儲存保存範本與詞庫
- 長圖匯出等待圖片載入並進行 Base64 預取，避免封面空白
- 支援一鍵重新整理系統範本/詞庫並保留使用者自訂資料
- `start.bat` 與啟動腳本相容性修復
- 瀏覽器自動開啟邏輯優化
